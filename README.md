## Install

With NPM

> npm install countries-flag

With Yarn

> yarn install countries-flag

## Use

CSS
```css
@import 'countries-flag'
```

Sass

```scss
@import 'countries-flag/index.scss';
```

HTML

```html
    <!-- China flag -->
    <span class='flag flag-cn'></span>
```
